# hyprfiles
hyprland 合成器下，桌面环境的一系列配置文件
# 配置说明

- 终端：Alacritty，Wezterm
- 应用启动器：Anyrun，Rofi
- 音量，亮度控制：Avizo （自带了一个修改音量亮度后的通知）
- 通知：Dunst
- 状态栏：Waybar
- 壁纸：Swww
- 锁屏：Gtklock

详细参考配置文件
